##1. Xml读取

xpath 常用规则

| 表达式   |           描述           |
| :------- | :----------------------: |
| nodename |  选取此节点的所有子节点  |
| /        | 从当前节点选区直接子节点 |
| //       |  从当前节点选取子孙节点  |
| .        |       选取当前节点       |
| ..       |   选取当前节点的父节点   |
| @        |         选取属性         |
